Bixie Slideshow for UIkit

A simple slideshow that uses the UIkit animation classes for animation. The slideshow simply shows and hides the divs that are in the container. UIkit does the rest!

Any suggestons, issues or contributions can be made on Github

Included CSS

For the navigation the addon "Dotnav" is used, so make sure you include that CSS/Less. 
<link rel="stylesheet" href="vendor/uikit/css/uikit.gradient.min.css">
<link rel="stylesheet" href="vendor/uikit/css/addons/uikit.gradient.addons.min.css">
All other CSS for the demosite is inline in the head.

Included Javascript

<script src="vendor/jquery/jquery-2.1.1.min.js"></script>
<script src="vendor/uikit/js/uikit.min.js"></script>
<script src="js/bixslideshow.js"></script>

General markup

In this example the container is styled with the uk-container uk-container-center uk-position-relative uk-height-1-1 classes. 
<div class="" data-bix-slideshow data-uk-observe>
    <div>....</div>
    <div>....</div>
    <div>....</div>
</div>

Javascript options

Option Possible value Default Description
delay integer 5000 Delay between slides
nav boolean true Show Dotnav-navigation in bottomleft corner
